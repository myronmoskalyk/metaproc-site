Welcome to the MetaProc docs. MetaProc is a button-driven, template-based app for
**frequentist meta-analysis** — in the browser or on the desktop — built on the
established R engines (`metafor`, `netmeta`, `meta`, `robvis`, `esc`).

## Start here

- **[Quickstart — your first meta-analysis](/docs/quickstart)** — go from a blank
  workspace to a forest plot, GRADE summary, and exported report in about ten minutes.
- **[Web vs desktop](/docs/web-vs-desktop)** — an honest comparison of the two ways to run MetaProc.
- **[Product manual (generated)](/docs/manual)** — the full, source-synced capability
  inventory. Do not hand-edit; it is regenerated from the app repository.

## The one idea to take away

Every result MetaProc shows you is produced by **building R code from your choices,
running that exact code, and showing you the same code**. What you see in the R code
panel is what ran — so any analysis you do here is auditable and reproducible outside
the app.
